# Podili Susmitha — Personal Portfolio

A premium, luxury-themed personal portfolio website built with **pure HTML5, CSS3, and Vanilla JavaScript** — no frameworks, no backend.

Created for **Web Development Internship Task-04** at **Prodigy InfoTech**.

## 🎨 Design Theme

**Luxury Nude Aesthetic** — cream, nude beige, sand, warm brown, dark brown, charcoal, and soft gold accents, paired with glassmorphism panels, soft shadows, and smooth rounded corners.

**Fonts:** Sora (display), Outfit (body), Poppins (accents) — loaded from Google Fonts.

## 📁 Project Structure

```
portfolio/
│── index.html          # Main HTML — all sections
│── style.css            # All styling, theme variables, animations
│── script.js             # All interactivity (vanilla JS, no libraries)
│── assets/
│   ├── images/           # Add project screenshots here
│   ├── icons/             # Add any custom icons here
│   └── resume.pdf        # Replace with your actual resume
│── README.md
```

## ✨ Features

- Animated typing effect in the hero (rotating role titles)
- Smooth scroll navigation with active-link highlighting
- Scroll-reveal (fade + slide up) animations throughout
- Animated counters and skill progress bars
- Glassmorphism cards, floating ambient background blobs
- Button ripple micro-interaction
- Fully responsive: desktop, laptop, tablet, and mobile
- Functional front-end contact form (client-side validation + success message — no backend, so no data is actually sent anywhere)
- Respects `prefers-reduced-motion` for accessibility

## 🛠️ How to Use

1. Unzip the folder.
2. Open `index.html` in any modern browser — that's it, no build step or server required.
3. Replace `assets/resume.pdf` with your real resume file (keep the same filename, or update the `href` in `index.html`).
4. Update social links (LinkedIn, GitHub, email) inside `index.html` — search for `href="#"` placeholders.
5. Swap the `.project-image` icon placeholders for real screenshots by adding `<img>` tags pointing to files inside `assets/images/`.
